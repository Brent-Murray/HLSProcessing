# HLSProcessing

## Overview

This repository provides tools and workflows for processing Harmonized Landsat and Sentinel-2 (HLS) data. The focus is on post-download processing, analysis, and transformation of HLS products.

This repository **does not cover data discovery or download workflows**. For those steps, please refer to the official HLS resources listed below.

## References

* [HLS User Guide](https://lpdaac.usgs.gov/documents/1698/HLS_User_Guide_V2.pdf)
* [HLS Data Resources GitHub](https://github.com/nasa/HLS-Data-Resources)

## Scope

This repository is intended for:

* Processing HLS data after acquisition
* Developing analysis pipelines
* Implementing transformations and feature extraction
* Best Available Pixel Compositing for HLS data

This repository does **not** include:

* Instructions for searching or downloading HLS datasets
* API access or data acquisition workflows

For downloading and accessing HLS data, please refer to the official [HLS Data Resources GitHub](https://github.com/nasa/HLS-Data-Resources).
